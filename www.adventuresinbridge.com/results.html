RESULTS - HAND RECORDS

Thanks to the following individuals for helping "automate" bridge in San Diego County:

1) Marvin French - Marvin introduced the first duplimate (board duplicating machine) in San Diego County.

2) Evan Bailey - Evan spent countless hours programming software that reads the formatting and often reformatting of the ACBLSCOR records. He produced an easy-to-read results page which includes overalls and a copy of the hand record and its results; the premier and "best" software in 2004.

3) Matthew Kidd - We are currently using Matt's ACBLMerge program. ACBLMerge integrates the game results file with the hand record file to produce overall game results with many preformatted files designed help bridge players; the "best" software in 2011.
